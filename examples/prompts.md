# Prompt Examples

The prompts below can be copied directly into Codex for use.

## Illustration planning only

```text
Use $ian-xiaohei-illustrations but don't generate images yet.
Please analyze the article below to identify where illustrations would add value, and output a shot list of about 5 images.
For each image, clearly specify:
- Which paragraph it follows
- The image's theme
- The core idea
- The structure type
- What Xiaohei is doing in the image
- Suggested elements
- Suggested English annotation words

<paste article>
```

## Illustrations for the article body

```text
Use $ian-xiaohei-illustrations to generate 4 surreal Xiaohei body illustrations for the article below.
Requirements: 16:9 landscape, pure white background, black hand-drawn line art, a few red/orange/blue English handwritten annotations.
Each image should convey only one core structure—no PPT-style infographics, no cute cartoon style.

<paste article>
```

## Illustration strategy for long-form articles

```text
Use $ian-xiaohei-illustrations to build an illustration strategy for this long article.
Don't distribute illustrations evenly—pick only the cognitive anchor points: core judgments, input/output loops, before/after comparisons, common pitfalls, and transition paths.
Default to 6-8 images; output the shot list first without generating any images.

<paste article>
```

## One image for a single point

```text
Use $ian-xiaohei-illustrations to generate one 16:9 body illustration for this point:

Trust isn't proclaimed—it's built piece by piece, one piece of evidence at a time.

The visual should be surreal but clean, and Xiaohei must carry out the core action.
Use at most 5 English annotations, and keep them short.
```

## Workflow theme

```text
Use $ian-xiaohei-illustrations to generate one image for "turning a single piece of raw material into three kinds of content: traffic, trust, and conversion."
Don't draw a formal flowchart, and don't recycle the old "one fish, multiple dishes" example.
Please reinvent a new low-tech metaphor, with Xiaohei taking part in the core action.
```

## Editing an image: remove the title

```text
Use $ian-xiaohei-illustrations to help me edit this image.
Remove the "Workflow / Flowchart" title and underline in the top-left corner; keep everything else unchanged.
Don't add any new text or objects.
```

## Editing an image: strengthen Xiaohei's involvement

```text
Use $ian-xiaohei-illustrations this image has the right direction, but Xiaohei feels a bit decorative.
Please keep the core idea unchanged and regenerate a version where Xiaohei truly drives the structure.
Make the visual a bit stranger, but still pure white, clean, and with minimal text.
```

## Generate a set of style samples

```text
Use $ian-xiaohei-illustrations to output 5 Xiaohei body illustrations on different themes.
Cover these themes respectively: information overload, product validation, content compounding, the one-person company, and building trust.
Generate each one separately—don't combine them into a single image.
```
